Privacy Policy

Last updated: May 12, 2026

Webflow Cleanup Assistant

Webflow Cleanup Assistant analyzes the currently opened webpage locally in the user's browser.

Data collection

The extension does not collect, store, transmit, sell, or share personal data or website content. All analysis is performed locally in the browser and no data is sent to external servers.

Website content

The extension temporarily reads the current webpage content, DOM structure, class names, and computed styles only when the user clicks the analyze button. This access is used only to generate the local cleanup report.

Permissions

The extension uses the activeTab permission to access the current page only after the user activates the extension. It uses the scripting permission to run the audit script on the current page.

Third parties

The extension does not use analytics, advertising, remote code, tracking scripts, or third-party data processors.
